# Obsidian TaskChute Mini

A small Obsidian plugin to reduce visual noise when using TaskChute-style workflows.

This plugin is designed to make daily notes and task lists feel lighter by minimizing elements that are useful for machines but distracting for humans.

⚠️ **This is an unofficial plugin.**  
This project is **not affiliated with, endorsed by, or officially related to TaskChute or its developers.**

---

## Features

- Reduces visual noise in TaskChute-style task management
- Keeps metadata while making it less noticeable
- Focuses on "doing" rather than "managing"

This plugin does **not** add new task logic or tracking features.  
It only adjusts how things look and feel.

---

## Installation (BRAT)

This plugin is intended to be installed via **BRAT**.

1. Install the BRAT plugin in Obsidian
2. Open BRAT settings
3. Add this repository URL:https://github.com/plzsayyes3/obsidian-taskchute-mini
4. Enable the plugin from Obsidian settings

---

## Manual Installation

1. Download `main.js` and `manifest.json` from the latest release
2. Create a folder:.obsidian/plugins/obsidian-taskchute-mini/
3. Put the files into that folder
4. Reload Obsidian and enable the plugin
